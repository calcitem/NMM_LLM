Build FullGame DB

Scans human-played games to find common positions, then BFS-expands around them to cover opponent responses. Output is a sorted binary (.bin) file. Script: tools/build_fullgame_db.py

Build Endgame Solved DB

Retrograde-solve all positions with ≤5 pieces per side. Produces endgame_{nW}_{nB}.wdl files. Takes minutes to hours. Script: tools/build_endgame_db.py

Endgame Self-Play

AI vs AI games starting from random endgame positions, for rapid EndgameDB enrichment. Script: tools/endgame_play.py

Self-Play

Run AI vs AI games to accumulate training data. Saves game records to data/games/. Script: tools/self_play.py

Train Value Network

Train an MLP value network from JSONL game records. Requires PyTorch. Output is a .npz weight file used by the neural leaf evaluator. Script: tools/train_value_net.py

Evolve Heuristic Weights

Era-aware (1+1)-ES to find better heuristic weights. Per-personality or gauntlet mode (all personalities). Script: tools/evolve_weights_v2.py

Gauntlet mode: evaluates each candidate against all personalities; threshold is 0.52. Saves to data/weights/best.json.

Database Paths

Configure paths to external database files. Changes take effect on next server restart.

Auto-Evolve After N Games

Server automatically runs gauntlet evolution after every N saved human games. 0 = disabled. Log: data/weights/auto_evolve.log

—

Name Openings

Merges duplicate openings and names unnamed ones via the LLM. Deduplication always runs; LLM naming requires Ollama. Script: tools/name_openings.py

⚠ Purge AI Self-Play Learning

Removes AI-generated openings and self-play game records. Keeps human games and book-imported openings. Writes a backup to data/backups/ before making changes.
