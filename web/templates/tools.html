Build FullGame DB

Scans human-played games to find common positions, then BFS-expands around them to cover opponent responses not seen in the corpus. Output is a sorted binary (.bin) file.

Build Endgame Solved DB

Retrograde-solve all positions with ≤5 pieces per side. Takes minutes to hours depending on depth. Output goes to data/endgame/.

Self-Play

Run AI vs AI games to accumulate training data. Uses tools/self_play.py. Saves game records to data/games/.

Evolve Heuristic Weights

Runs an era-aware (1+1)-ES to find better heuristic weights. Uses tools/evolve_weights_v2.py.

Gauntlet mode evaluates against all personalities and saves to data/weights/best.json. Threshold is 0.52 (vs 0.55 per-personality).

Auto-Evolve After N Games

Server automatically runs gauntlet evolution after every N saved human games. Set to 0 to disable. Log goes to data/weights/auto_evolve.log.

Name Openings

Merges duplicate openings and names unnamed ones via the LLM. Requires Ollama to be running for naming; deduplication always runs.

⚠ Purge AI Self-Play Learning

Removes AI-generated openings and self-play game records. Keeps human games and book-imported openings. Writes a backup before making changes.
